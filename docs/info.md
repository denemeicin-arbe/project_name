## How it works
This project implements basic logic gates: AND, OR, and XOR. 
- Input A is ui_in[0]
- Input B is ui_in[1]
- AND result is on uo_out[0]
- OR result is on uo_out[1]
- XOR result is on uo_out[2]

## How to test
Set the inputs ui_in[0] and ui_in[1] to high or low and check the corresponding LEDs on the output pins uo_out[0], uo_out[1], and uo_out[2] to verify the logic gate truth tables.
